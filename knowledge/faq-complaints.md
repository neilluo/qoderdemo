# 海底捞客服FAQ - 投诉与售后

## 食品安全类
- **异物投诉**：立即致歉 → 当餐免单 → 赠送200元代金券 → 48h内店长回访
- **食材不新鲜**：立即更换 → 该菜品免费 → 赠送果盘 → 记录并上报品控
- **过敏反应**：立即协助就医 → 全额承担医疗费 → 总部品控介入

## 服务类
- **等位时间过长**：超过预估时间15分钟 → 赠送小食拼盘 → 优先安排
- **服务态度差**：致歉 → 更换服务人员 → 赠送菜品 → 店长跟进培训
- **上菜慢**（超过15分钟）：致歉 → 催单加急 → 赠送一份小菜

## 账单类
- **多收费**：核实后立即退差额 → 赠送50元代金券致歉
- **优惠券未生效**：核实券规则 → 符合条件则手动补扣 → 不符合则解释规则
- **积分未到账**：确认结账手机号 → 48h内补录 → 如超时则手动补发

## 退款标准
| 场景 | 处理方式 | 时效 |
|------|---------|------|
| 外卖未送达 | 全额退款 | 即时 |
| 外卖缺餐 | 缺失菜品退款+5元补偿 | 24h内 |
| 团购券未使用 | 原路退款 | 1-3个工作日 |
| 堂食投诉退款 | 视情况部分/全额退款 | 当场处理 |

## 话术模板
- 开场："您好，非常抱歉给您带来不好的体验，我是海底捞客服小X，很高兴为您服务。"
- 致歉："对于这次的问题我们深感抱歉，这不是我们应有的服务标准。"
- 解决："为了弥补您的损失，我们将为您[具体补偿措施]，请问您看可以吗？"
- 结束："感谢您的理解和支持，祝您生活愉快！如有任何问题随时联系我们。"
